# Python Ideal Function Assignment

This project implements a Python-based analytical system to identify ideal mathematical functions that best approximate given training datasets and to map test observations to the selected ideal functions.

## Features

- Least squares based ideal function selection
- Test data mapping using deviation threshold
- SQLite database integration
- Interactive visualization using Bokeh
- Object-oriented architecture
- Unit testing with Python unittest
- Git version control workflow
